## 介绍
Sight Vue UI 是一个基于 Vue.js 的 UI 组件库。现在还处于开发阶段，请不要在生产环境中尝试使用。

![Sight](https://i.vimeocdn.com/video/322403573_1280x720.jpg)

## 开始使用

```bash
npm install sight-vue
```
或`
```bash
yarn add sight-vue
```
