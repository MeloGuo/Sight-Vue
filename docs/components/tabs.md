## 简单用法

</ClientOnly>

**代码**

```html
<s-tabs :selected="selected">
  <s-tabs-head>
    <s-tabs-item name="1">1</s-tabs-item>
    <s-tabs-item name="2">2</s-tabs-item>
  </s-tabs-head>
  <s-tabs-body>
    <s-tabs-pane name="1">content 1</s-tabs-pane>
    <s-tabs-pane name="2">content 2</s-tabs-pane>
  </s-tabs-body>
</s-tabs>
```
